Acquisition Warrant Update

To update a warrant, ensure that you received a memorandum to Update Acquisition Warrant Information from the CSS.
  1. Go to milPDS: Relocations and Employment > Personnel Employment > Employment > Assignment Actions
    Another option is to Press Ctrl+L, then type in %lvl and the ASSIGNMENT INFORMATION should pop up
  2. Find the member's info (Press F11, type in member's SSN or name%, then Ctrl+F11)
  3. Select "Assignment" button at the bottom
  4. Select "Extra Information" at the bottom
  5. Select the "ACQ CONTRCTNG LIMIT" tab and then an empty Details tab at the bottom
  6. Fill in all the following information, which should've been provided on the memorandum you were provided:
    Start Date
    End Date
    Warrant Type
    Warrant Limit
    Contracting Job Site (Y/N)
    then OK.
  7. Save the data.
